# Meridian — Financial Planner

A scenario-based personal financial planning application built with React, TypeScript, and Recharts.

## Features

- **Budget Snapshot** — View and edit your income, fixed and variable expenses, spending breakdown by category, and a 6-month spending trend chart.
- **Scenario Comparison** — Compare the 10-year financial outcome of continuing to rent vs. purchasing a home. Adjust home price, down payment, mortgage rate, and appreciation with real-time sliders.
- **Timeline Projection** — Project your savings growth over time across three scenarios (Conservative, Moderate, Optimistic) with inflation-adjusted real returns.
- **Goal Tracker** — Track multiple financial goals (emergency fund, down payment, retirement, etc.), analyze contribution sensitivity, and visualize your path to each goal.

## Tech Stack

- [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) — build tooling
- [Recharts](https://recharts.org/) — all charts and visualizations
- [Lucide React](https://lucide.dev/) — icons
- [Tailwind CSS](https://tailwindcss.com/) — utility-first styling foundation

## Setup

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.

## Sample Data

The pre-loaded sample data reflects a **$410,000 gross income household in New York City**, including realistic rent, transportation, dining, and lifestyle expenses for the area.
